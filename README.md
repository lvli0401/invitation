# invitation

A invitation page project for Broccoli & Co.

### usage:

1. `git clone https://github.com/lvli0401/invitation.git <dir-name>`  
2. `cd <dir-name>`
3. `yarn` or `cnpm i` or `npm i`

### run in dev:

`yarn` or `npm run start`

### build:

`yarn build` or `npm run build`
